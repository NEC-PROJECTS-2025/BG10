Overview

The Glaucoma Detection Project is a cutting-edge initiative designed to tackle one of the leading causes of irreversible blindness. By leveraging advanced machine learning and deep learning technologies, the project aims to provide accurate and early detection of glaucoma using retinal fundus images.

Key Objectives

  • To enable early detection of glaucoma using AI-driven technologies.
  • To improve diagnostic accuracy, precision, and recall rates with integrated frameworks.
  • To enhance ophthalmic healthcare accessibility with automated diagnostic systems.

Features and Achievements

Our project combines powerful convolutional neural networks like ResNet50 and VGG-16 with Random Forest classifiers, ensuring high accuracy and reliability in glaucoma detection.

Technology Stack

We utilize advanced technologies and methodologies to achieve our objectives:

  • Deep Learning Models: ResNet50, VGG-16.
  • Machine Learning Classifier: Random Forest.
  • Feature Extraction: Gray-Level Co-occurrence Matrix (GLCM).
  • Preprocessing: RGB to Grayscale conversion for improved texture analysis.
